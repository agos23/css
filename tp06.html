TP06-CSS

1.1- ¿Qué significa CSS?

CSS significa en español: Hojas de Estilo en Cascada y en ingles significa Cascading Style Sheets.

1.2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

El CSS sirve para darle un estilo /diseño a un sitio web o pagina,

1.3- ¿A qué se llama regla de CSS?

Se llama regla CSS a un conjunto de propiedades y valores que son aplicadas en varias etiquetas segun cual se este utilizando.

1.4- ¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

Seleccionador:
el seleccionador selecciona elementos para darle un estilo .
Declaracion:
La declaracion le aplica propiedades al elemento que fue seleccionado con el selector ,solo tiene una regla pero puden haber mas.
Propiedades:
estiliza la html ,ejemplo la propiedade color la cual le da el color que elijas a las letras del parrafos (cada valor es ubicado a la izquierda).
Valor de la propiedad:
deven tener un valor. cada una de la propiedades tine otorgado valores definidos (cada valor es ubicado a la derecha).

1.5- ¿Qué es un selector en CSS?

Un selector en CSS es aquel que selecciona elementos y permite que le demos un estilo .podemos usarlo para diseñar diferrentes elementos.

1.6- Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

Selector ID :sirve para un elemento ,ya que la ID es unica para cada elemento. Selector de Clase:una clase puede aparecer varias veces en una pagina. Selector Universal:Nos permite aplicar estilos a todos los elementos HTML de un documento. Selector de Etiquetas:Aplicamos propiedades CSS a una etiqueta. Selector Decendente:seleccionamos un elemento que está dentro de otro del mismo tipo. Estos están formados por dos o más selectores separados entre si. El último selector será el elemento HTML al que se le aplicará las propiedades de CSS.
